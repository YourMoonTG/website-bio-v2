Обзор проекта

Кастомный Linux дистрибутив — это полностью автоматизированная система настройки Arch Linux WSL для Windows 10/11. Проект включает в себя PowerShell скрипты для автоматической установки, настройки среды разработки и создания оптимальной рабочей среды.

Ключевые возможности:

  • Автоматическая установка и настройка Arch Linux WSL
  • Создание пользователя Moon с правами sudo
  • Настройка zsh с oh-my-zsh и powerlevel10k
  • Установка современных инструментов разработки
  • Конфигурация WSL для оптимальной производительности
  • Система логирования и мониторинга установки
  • Поддержка как новых, так и существующих установок
